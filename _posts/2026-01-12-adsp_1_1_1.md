# 데이터

## 정의

- 라틴어 ‘dare(주다)’, ‘Datum(주어진 것, 선물, 자료)’
- 연구나 조사의 바탕이 되는 재료 혹은 자료

## 특성

- 데이터
    - 객관적인, **있는 그대로의 사실**
    - 가공되지 않은 자료
    - ex) 수학 80점, 영어 100점 등

- 정보
    - 데이터에서 얻은 것으로부터 **가공된** 자료
    - ex) 수학과 영어의 평균 점수는 90점

- 존재적 특성 vs 당위적 특성
    - “존재적 특성”: **있는 그대로의 사실**을 나타내는 것
        - **“데이터”**의 특성

    - “당위적 특성”: 추론 , 예측, 전망, 추정을 위한 정보의 **근거**가 될 수 있는 것
        - **“정보”**의 특성



## 유형

### 정성적 데이터 vs 정량적 데이터

- 정성적 데이터
    - 기준이 명확하지 않은 데이터
    - ex) 언어, 문자 등 / 정성적 데이터는 집합으로 표현될 수 없음

- 정량적 데이터
    - 기준이 명확한 데이터
    - ex) 수치, 도형, 기호 등(30cm, 정육면체, 3시 방향) / 집합으로 표현 가능한 데이터


### 정형 데이터 vs 비정형 데이터 vs 반정형 데이터

- 정형 데이터
    - 고정된 틀이 있음 & 연산이 가능한 데이터
    - 관계형 DB에 주로 저장, 데이터의 수집 및 관리 용이

- 비정형 데이터
    - 고정된 틀이 없음 → 연산도 불가능
    - NoSQL에 저장, 데이터의 수집 및 관리가 어려움

- **반정형 데이터**
    - **고정된 형태는 있음 & 연산은 불가능**
    - 테이블보다는 **파일 형태**로 저장 → 가공을 거쳐 정형 데이터로 변환 가능


### 암묵지와 형식지

- 암묵지(Tacit Knowledge)
    - 학습 및 체험을 통해 **개인으로 습득**되어 있음 but **겉으로는 드러나지 않는** 상태
    - 머릿속에는 존재, 언어 및 문자 등을 통해 표현되지만 않음
    - 경험을 통해 체득하는 경우가 다반사(시행착오 등)

- 형식지(Explicit Knowledge)
    - **외부로 표출된 암묵지**
    - 형태: 문서, 메뉴얼 등
    - **여러 사람이 공유 가능**한 지식
    - 어떤 형태로든 형상화된 지식이면 형식지라 함(교과서, 데이터베이스, 신문, 비디오 등)

- 암묵지와 형상지의 상호작용
    - 공유되기 어려운 암묵지가 형식지로 표출 및 연결되면 → 그 상호작용으로 지식이 형성됨
        - 암묵지는 그 특성상, 공유되기 어려움 → 개인에게 **내면화** 되어있기 때문<br>(암묵지 - 내면화)
        - “개인에게 **내면화**된 암묵지 → 조직의 지식으로 **공통화”**를 위해 **표출화**됨<br>(암묵지 - 공통화, 형식지 - 표출화)
        - 표출화된 지식 → 다시 개인의 지식으로 연결하는 **연결화** 과정<br>(형식지 - 연결화)
        - 이렇게 연결된 지식은, 다시 개인에게 **내면화**됨<br>(암묵지 - 내면화)

# 데이터와 정보

## DIKW 피라미드

### 구성요소: 데이터 / 정보 / 지식 / 지혜

- 데이터(**D**ata): 객관적인 사실
    - 개별 데이터 자체는 그 의미가 크게 중요하지 않음

- 정보(**I**nformation): 객관적인 사실로부터 의미가 도출된 것
    - 데이터의 가공, 처리 및 데이터 간 연간관계로부터
    - 정보가 내포하는 의미는 유용하지 않을 수도 있음

- 지식(**K**nowledge)
    - 정보 구조화 → 유의미한 정보 분류, 개인적 경험 결합 → 고유의 지식으로 내재화

- 지혜(**W**isdom): 지식의 축적 + 아이디어, 창의적 산물

### DIKW 피라미드


![](/images/2026-01-12-adsp_1_1_1/image1.png)

- **지혜(W)**
    - A의 다른 물건도 저렴할 것

- **지식(K)**
    - 더 저렴한 A로부터 연필을 사야겠다

- **정보(I)**
    - A 연필이 더 저렴하다

- **데이터(D)**
    - A는 1000원, B는 2000원에 연필 판매


## 데이터 관련 상식

### 비트 / 바이트

- 비트(bit)
    - 신호의 최소 단위
    - 0과 1 두 가지 값으로 표현
    - binary digit(이진수)의 약자

- 바이트(byte)
    - 데이터의 양을 나타내는 단위
    - 8개의 비트로 구성
    - 1바이트 → 숫자, 영어의 한 글자 표현 가능한 크기(한글은 한 글자당 2바이트 크기)


### 데이터의 단위

- 1바이트 = 8비트
- 1킬로바이트(KB) = 1024바이트
- 1메가바이트(MB) = 1024킬로바이트
- 1기가바이트(GB) = 1024메가바이트
- 1테라바이트(TB) = 1024기가바이트
- 1페타바이트(PB) = 1024테라바이트
- 1엑시바이트(EB) = 1024페타바이트
- 1제타바이트(ZB) = 1024엑시바이트
- 1요타바이트(YB) = 1024제타바이트
