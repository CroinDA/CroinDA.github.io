앞으로 공부할 때 문제가 생겼던 점들을 하나씩 기록해 나가볼 생각이다.
기록해두면, 추후에 나도 동일한 문제를 반복하지 않을 수 있을 것이고 이 글을 혹시 볼 지 모르는 다른 개발자 분들에게도 도움이 될 수 있겠다는 생각에 시작하게 되었다.<br><br>
오늘은 그 첫번째로, GitHub 사용 시 GPG 서명을 남길 수 있도록 설정하는 방법에 대해 기록해 보려고 한다.
Local에서 IDE(VSCode, Cursor 등)로 GitHub 사용 시 Commit 마다 Verified 문구가 붙을 수 있도록 GPG Key를 발급 받아 등록하는 과정을 남겨두어, 많은 분들이 도움을 받을 수 있었으면 좋겠다.<br><br>
